# Périmètre du projet

Ce projet (**BUT SD RAG Assistant**) est un assistant RAG non officiel pour le BUT Science des Données, centré sur l'IUT de Metz (Université de Lorraine) et le référentiel national (PPN).

## Sources autorisées

- Programme Pédagogique National (PPN) BUT Science des Données, publié par le ministère de l'Enseignement supérieur.
- Pages publiques de l'IUT de Metz / Université de Lorraine (présentation, parcours, contacts, flyers).
- Fiches publiques tierces officielles ou institutionnelles (Onisep, Parcoursup) décrivant la formation.

## Sources interdites

- Documents ARCHE (ENT, plateforme pédagogique interne).
- Supports de cours privés, polycopiés de profs.
- Grilles de coefficients ou de notation internes non publiées.
- Notes, bulletins, données personnelles d'étudiants.
- Mails, documents confidentiels, cas individuels.

## Règle de fiabilité

Si une information n'est pas disponible dans les sources publiques intégrées, l'assistant doit le dire explicitement et renvoyer vers ARCHE, l'IUT ou le responsable pédagogique. Aucune invention, aucun exemple fictif pour les coefficients, notes, SAÉ internes ou modalités d'évaluation.

## Mention obligatoire

Le projet est un outil pédagogique non officiel. Les réponses concernant des modalités précises (coefficients, dates, contacts administratifs) doivent toujours être vérifiées auprès des sources officielles.
